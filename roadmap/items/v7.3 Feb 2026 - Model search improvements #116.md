# Model search improvements

Incremental improvements to the new [django-modelsearch](https://github.com/wagtail/django-modelsearch) re-implementation of Wagtail search backends.

## Intended outcome

Tentatively including:

- Vector indexing support
- Search permission handling
- Filtering on related objects
